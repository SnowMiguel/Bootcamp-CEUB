# PortfolioHUB - Meus Projetos Profissionais e Acadêmicos
Este repositório de portifólio foi criado para o desenvolvimento e demonstração de habilidades adiquiridas durante a graduação em Engenharia de Software. Ele é direcionado especialmente à matéria Bootcamp

##  Estrutura do Repositório
- **Projetos Acadêmicos**: Projetos voltados a atividades e entregas próprias da matéria
- **Projetos Pessoais**: Iniciativas individuais e experimentos técnicos.

##  Projeto em Destaque (Página Web/Portifólio)
Uma aplicação web estática construída para demonstrar conceitos de versionamento e expor as minhas expertises e qualificações a titulo de currículo profissional.
