## Projeto em Destaque (Página Web/Portfólio)
Uma aplicação web estática construída para demonstrar conceitos de versionamento e expor expertises e qualificações como currículo profissional.
*   **Tecnologias Utilizadas:** HTML5 (98%) e CSS3 (2%) para estruturação e estilização visual.
*   **Funcionalidades:** Biografia, currículo detalhado e exibição de competências.
